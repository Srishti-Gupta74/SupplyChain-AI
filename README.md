# SupplyShield AI
**Adaptive Conformal Risk Triage for Resilient Supply Chains**

> "Traditional systems answer *what will demand be?*  
> SupplyShield AI answers *how much should you trust that forecast?*"

---

## Quick Start

```bash
pip install -r requirements.txt
python main.py          # trains models, runs evaluation, saves outputs/
streamlit run dashboard/app.py   # launch Risk Command Center
```

---

## Using Real M5 Data (instead of synthetic)

1. Download from Kaggle: https://www.kaggle.com/competitions/m5-forecasting-accuracy
2. Place files in `data/`:
   - `sales_train_validation.csv`
   - `calendar.csv`
   - `sell_prices.csv`
3. Edit `config.yaml`:
   ```yaml
   data:
     mode: m5    # change from 'synthetic'
     n_skus: 100
   ```
4. Re-run `python main.py`

## What It Does

| Component | What it solves |
|---|---|
| LightGBM base forecaster | Point demand prediction |
| Quantile Regression | Non-conformal UQ baseline |
| Static Conformal (MAPIE) | Principled UQ, assumes stationarity |
| **ACI (Gibbs & Candès 2022)** | **Adaptive UQ — stays honest during demand shocks** |
| Forecast Reliability Score | Translates calibration quality → 0–100 for operators |
| Newsvendor Safety Stock | Derives SS directly from ACI interval width |
| Risk Command Center | Streamlit dashboard — one screen, zero stat expertise needed |

---

## Key Results (Synthetic Data, 50 SKUs)

| Method | Coverage (All) | Coverage (Volatile) |
|---|---|---|
| Quantile Regression | 87.9% | 82.2% |
| Static Conformal | 94.1% | **73.2%** ← collapses during shocks |
| **ACI** | **95.6%** | **76.9%** |

**Underestimation audit:** ACI still shows volatile-period gap — this is the open research frontier ACI partially addresses. The audit flag triggers automatic escalation in the decision layer.
